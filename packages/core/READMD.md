## PPResume Core

The PPResume Core is the core engine of [PPResume](https://ppresume.com). It is
in charge of the typesetting and layout of the resume.

### Installation

- npm

```bash
npm install @ppresume/core
```

- pnpm

```bash
pnpm add @ppresume/core
```

- yarn

```bash
yarn add @ppresume/core
```

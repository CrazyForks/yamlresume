## YAMLResume CLI

The YAMLResume CLI is a command-line interface for
[YAMLResume](https://yamlresume.dev).

### Installation

- npm

```bash
npm install -g yamlresume
```

- pnpm

```bash
pnpm add -g yamlresume
```

- yarn

```bash
yarn global add yamlresume
```
